# My Portfolio Vault

> My personal source of truth for everything career-related.
> Raw data lives here. Tailored documents are generated *from* here.

---

## How this works

```
01_master_profile/   → Who I am, skills, education, awards — the evergreen stuff
02_projects/         → One folder per project, written in detail
03_outputs/          → Generated CVs, resumes, cover letters (never the source of truth)
```

**The rule:** Never edit a resume directly. Edit the source files here, then regenerate.

## Workflow: creating a new CV or resume

1. Identify the role/audience
2. Pick 4–6 projects from `02_projects/` that are most relevant
3. Pull the matching skills from `01_master_profile/skills.md`
4. Draft the output in `03_outputs/resumes/` or `03_outputs/cvs/`
5. Name it clearly: `YYYY-MM_role_company.md`

---

## Keeping this up to date

| Trigger | Action |
|---|---|
| Finish a project | Add a folder to `02_projects/` |
| Get a win, compliment, or result | Add a line to `brag_sheet.md` |
| Learn a new skill | Update `skills.md` |
| Get a testimonial or reference | Add to `testimonials.md` |
| Apply for a role | Create a new file in `03_outputs/` |
